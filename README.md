# 💰 FundsFlow API

> A modern personal finance management API designed to help users track their money, manage budgets, build savings habits, and gain better control over their financial future.

![Status](https://img.shields.io/badge/status-core%20features%20complete-brightgreen)

---

## 📌 About The Project

FundsFlow is a production-style backend application that helps users understand and manage their finances.

The goal of this project is to build a secure, scalable, and intelligent financial management system where users can:

* Track income and expenses
* Create and monitor budgets
* Set savings goals
* Analyze spending habits
* Receive financial insights

FundsFlow is being built with real-world backend engineering practices including authentication, database design, API documentation, testing, and deployment.

## 🚀 Features

### 👤 User Management ✅

* [x] User registration
* [x] Secure login system
* [x] Password encryption
* [x] JWT authentication
* [x] Protected API routes

---

### 💳 Transaction Management ✅

Users can:

* [x] Add income
* [x] Add expenses
* [x] Categorize transactions
* [x] View transaction history
* [x] Edit and delete transactions

Example:

```
Income:
Salary - ₦300,000

Expense:
Food - ₦20,000
Transport - ₦10,000
```

---

### 📊 Budget Management ✅

Create budgets and control spending.

Example:

```
Food Budget:
₦50,000/month

Transport Budget:
₦20,000/month
```

The system tracks:

* [x] Amount spent
* [x] Remaining budget
* [x] Spending percentage
* [x] Budget warnings

---

### 🎯 Savings Goals ✅

Users can create saving targets.

Example:

```
Goal:
Buy Laptop

Target:
₦800,000

Current Savings:
₦200,000
```

FundsFlow helps users monitor progress toward their goals.

### 📈 Financial Analytics ✅

The API provides financial insights such as:

* [x] Most used spending categories
* [x] Spending trends
