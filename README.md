# 💰 FundsFlow API

> A modern personal finance management API designed to help users track their money, manage budgets, build savings habits, and gain better control over their financial future.

![Status](https://img.shields.io/badge/status-in%20development-orange)

---

## 📌 About The Project

FundsFlow is a production-style backend application that helps users understand and manage their finances.

The goal of this project is to build a secure, scalable, and intelligent financial management system where users can:

* Track income and expenses
* Create and monitor budgets
* Set savings goals
* Analyze spending habits
* Receive financial insights

FundsFlow is being built with real-world backend engineering practices including authentication, database design, API documentation, testing, and deployment.

---

## 🚀 Features

### 👤 User Management

* User registration
* Secure login system
* Password encryption
* JWT authentication
* Protected API routes

---

### 💳 Transaction Management

Users can:

* Add income
* Add expenses
* Categorize transactions
* View transaction history
* Edit and delete transactions

Example:

```
Income:
Salary - ₦300,000

Expense:
Food - ₦20,000
Transport - ₦10,000
```

---

### 📊 Budget Management

Create budgets and control spending.

Example:

```
Food Budget:
₦50,000/month

Transport Budget:
₦20,000/month
```

The system will track:

* Amount spent
* Remaining budget
* Spending percentage
* Budget warnings

---

### 🎯 Savings Goals

Users can create saving targets.

Example:

```
Goal:
Buy Laptop

Target:
₦800,000

Current Savings:
₦200,000
```

FundsFlow will help users monitor progress toward their goals.

---

### 📈 Financial Analytics

The API will provide financial insights such as:

* Monthly income
* Monthly expenses
* Saving rate
* Most used spending categories
* Spending trends
